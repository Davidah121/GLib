Construct a new LCG (Linear Congruential Generator) for random numbers. Generates psudo random numbers using the seed, multiplier, increment, and modulus value. Fast but not secure due to repeated values. The functions period is based on the parameters.

Parameters
seedThe starting seed. Default is 65535
multiplierThe multiplier Default is 48271
incrementThe increment Default is 0
modThe modulus value to use. Default is the maximum of a unsigned integer.
