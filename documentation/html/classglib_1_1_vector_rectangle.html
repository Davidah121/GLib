◆ VectorRectangle()

glib::VectorRectangle::VectorRectangle ( )

Construct a new VectorRectangle object Follows the SVG format for rectangles. 

XML format:
(Note that rx and ry may not appear)
    <rect x="" y="" rx="" ry="" width="" height="" />
    x = x of top left corner
    y = y of top left corner
    rx = x radius of the corners
    ry = y radius of the corners
    width = width
    height = height

NOTE: Fill rule doesn't affect this object Line cap doesn't affect this object !! Line join does affect this object !!
