The base Class object. Its purpose is to provide a more object oriented feel similar to Java. It stores a class name, and parent classes and can be accessed from individual objects during run time.

Parameters
nameThe name of the class. It should be unique.
parentClassesThe list of parent classes for the current class. It should be a reference to the class object.
