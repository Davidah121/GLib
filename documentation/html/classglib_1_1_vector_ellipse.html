◆ VectorEllipse()

glib::VectorEllipse::VectorEllipse ( )

Construct a new VectorEllipse object Follows the SVG format for an ellipse. 

XML Format:
    <ellipse cx="" cy="" rx="" ry=""/>
    cx = center x of the ellipse
    cy = center y of the ellipse
    rx = x radius
    ry = y radius

NOTE: Fill rule doesn't affect this object Line cap doesn't affect this object Line join doesn't affect this object
