◆ CombinationShape()

glib::CombinationShape::CombinationShape ( )

Construct a new Combination Shape object Contains a list of shapes that make up the current shape. If either shape collides with something, it is considered a collision.

Note that this object also contains a position, scale, and rotation. These will be applied before the shapes transforms are.
