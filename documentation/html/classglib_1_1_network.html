Construct a new Network object Allows network connection using TCP or UDP. Uses a separate thread to maintain the connection and check for message arrival.

Parameters
typeServers can have multiple connections where a client can only have one connection. valid types are: TYPE_SERVER TYPE_CLIENT
portMust be a valid port between [0,65535]
ipaddressMust be a valid ipv4 address. For clients, it is the IP to connect to. For servers, it is the IP that will receive connections.
amountOfConnectionsAllowedDefault is 64.
TCPIf set to false, UDP is used. Default is true
