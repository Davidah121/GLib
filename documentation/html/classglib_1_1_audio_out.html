static void glib::AudioOut::init ( unsigned int  deviceID = -1)
static

Opens a device specified by the deviceID value to output audio too. Spawns a thread to process that audio and output audio. Can not be recalled until dispose() is called and successfully returned.

Parameters
deviceIDThe deviceID should be obtained from the operating system. The default value is -1 which will map to the default output device.
Exceptions
AudioOutInitErrorError is thrown if the output device could not be initialized

static void glib::AudioOut::setBuffers ( int  b)
static

Set the amount of buffers for playing audio. The default is 2 buffers. Must be set before calling init.

Parameters
bThe number of buffers allows for decreases in the size of the buffers and keeps the OS fed with data to reduce studdering. The current value is set to reduce studdering and keep latency good.

◆ setSizeOfBuffer()

static void glib::AudioOut::setSizeOfBuffer ( int  b)
static

Sets the size of the individual buffers. The default is 2205 samples. Must be set before calling init.

Parameters
bThe size of the buffers in samples. The larger the size, the more milliseconds of audio that must be prepared at a time. The value 2205 was selected based on 2 buffers, stereo audio, 16 bit samples, and 44100 samples per second. This results in 50 milliseconds of audio latency with no studdering.
