)

Construct a new SimpleWindow object.

Parameters
titleThe title of the window.
widthThe width of the window. By default, it is 320px wide
heightThe height of the window. By default, it is 240px high
xThe x location of the top left part of the window. By default, it is -1 which will set the center x of the window at the center x of the main monitor.
yThe y location of the top left part of the window. By default, it is -1 which will set the center y of the window at the center y of the main monitor.
windowTypeThe type of window to create. The window type and window management can be combined using the logical or ( '|' ). Valid arguments are: NORMAL_WINDOW BORDERLESS_WINDOW TYPE_FOCUSABLE TYPE_NONFOCUSABLE TYPE_USER_MANAGED TYPE_THREAD_MANAGED Default value is 0 which is a Normal Focusable Thread Managed Window.
