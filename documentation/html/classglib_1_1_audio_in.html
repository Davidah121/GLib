◆ init()

static void glib::AudioIn::init ( unsigned int  device = -1)
static

Opens a device specified by the deviceID value to get audio input from. Spawns a thread to retrieve the audio. Can not be recalled until dispose() is called and successfully returned.

Parameters
deviceIDThe deviceID should be obtained from the operating system. The default value is -1 which will map to the default input device.
Exceptions
AudioInInitErrorError is thrown if the input device could not be initialized

Set the amount of buffers for playing audio. The default is 2 buffers. Must be set before calling init.

Parameters
