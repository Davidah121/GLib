Solves for the Arc Length of the Bezier curve from start time to end time. Note for all bezier curves above degree 2 (not a line) the arc length is approximated using Simpson's rule. The Bezier curve is divided into segments and each segement is approximated and added together to get the total arc length.

◆ getSimpleDerivativeAt()

Vec2f glib::BezierCurve::getSimpleDerivativeAt ( double  time)

Solves for the simple derivative of BezierCurve for the given time. Note that any time value can be used even if outside of the bounds [0,1].

The simple derivative is just the next point minus the previous point where time determines which points to use. This is useful if you just need to know if the function is increasing at the time given instead of the actual derivative.

Parameters
timeThe time value to solve for.
Returns
Vec2f returns the simple derivative as a Vec2f.
