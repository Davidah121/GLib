◆ VectorLine()

glib::VectorLine::VectorLine ( )

Construct a new VectorLine object Follows the SVG format for a line. 

XML Format:
    <line x1="" x2="" y1="" y2=""/>
    x1 = start x
    x2 = end x
    y1 = start y
    y2 = end y

NOTE: Fill rule doesn't affect this object !! Line cap does affect this object !! Line join doesn't affect this object
