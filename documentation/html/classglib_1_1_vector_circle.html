◆ VectorCircle()

glib::VectorCircle::VectorCircle ( )

Construct a new VectorCircle object Follows the SVG format for circles.

XML Format: <circle cx="" cy="" r=""> cx = center x of the circle cy = center y of the circle r = radius NOTE: Fill rule doesn't affect this object Line cap doesn't affect this object Line join doesn't affect this object
