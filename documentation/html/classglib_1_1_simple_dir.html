◆ deleteResource()

void glib::SimpleDir::deleteResource ( int  index)

Attempts to delete the reference at the index. The function may fail to delete the resource. If USE_EXCEPTIONS is defined, an DeleteError may be thrown. If USE_EXCEPTIONS is defined, an OutOfBoundsError may be thrown.

Note that deleting in this way will not move the data to the recycle bin.
